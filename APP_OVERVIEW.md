# 📱 Финтех Мобильное Приложение - Обзор

## 🎯 Общее описание

Это современное финтех мобильное приложение (Progressive Web App) для управления финансами, построенное на React с TypeScript. Приложение поддерживает два языка (русский и узбекский), две темы оформления (светлая и темная), и оптимизировано для мобильных устройств с базовым размером 430x932px.

---

## 🎨 Дизайн система

### Цветовая палитра
- **Основной цвет:** Фиолетовый (#7c3aed)
- **Градиенты карт:** Фиолетово-синий (от #667eea до #764ba2)
- **Доход:** Зеленый (#22c55e) с иконкой Plus
- **Расход:** Красный (#ff4757) с иконкой Minus
- **Фоновые цвета:**
  - Темная тема: #0f1420 (background), #1e2337 (cards)
  - Светлая тема: #f5f6fa (background), #ffffff (cards)

### Иконки
Все иконки из библиотеки **Tabler Icons** (https://tabler.io/icons):
- IconPlus - доход
- IconMinus - расход
- IconStar - цели (бывш. "Сбережения")
- IconCreditCard - карты
- IconChartPie - статистика
- И другие...

### Элементы UI
- **Скругление:** Все элементы списков с `rounded-full` (полностью круглые углы)
- **Анимации:** Использование Motion (бывш. Framer Motion) для плавных переходов
- **Адаптивность:** Пропорциональное масштабирование под любой размер экрана

- Цели → `/savings`

## 🌐 Локализация

### Поддерживаемые языки:
- **Русский (rus)** - по умолчанию
- **Узбекский (uzb)**

### Где выбирается:
1. На странице Onboarding (при первом входе)
2. В настройках профиля (ProfileSettings)

### Что переводится:
- Все тексты интерфейса
- Названия категорий
- Уведомления
- Даты и время
- Названия сервисов
- Ошибки и сообщения

### Реализация:
Использует Context API (LanguageContext):
```typescript
const { language, setLanguage } = useLanguage();
const content = {
  rus: { ... },
  uzb: { ... }
};
```

## 🎨 Темы оформления

### Поддерживаемые темы:
- **Темная (dark)** - по умолчанию
- **Светлая (light)**

### Где переключается:
- В настройках профиля (ProfileSettings)
- Переключатель рядом с выбором языка

### Что меняется:
- Цвет фона страницы
- Цвет фона карточек
- Цвет текста (основной и вторичный)
- Цвет границ
- Цвет иконок

### Реализация:
Использует Context API (ThemeContext):
```typescript
const { theme, setTheme, colors } = useTheme();
// colors содержит:
// - background
// - cardBackground
// - text
// - textSecondary
// - border
```
