Your task is to analyze the provided Figma design file and convert it into a complete Figma Make project structure.

1. Analyze Design Structure
- Identify all pages and frames
- Determine application sections and navigation structure
- Detect reusable UI components
- Identify layout grids and spacing systems

2. Extract Design System
Create a design system based on the file:
- Colors
- Typography
- Buttons
- Inputs
- Cards
- Modals
- Dropdowns
- Icons
- Navigation elements

3. Identify Components
Convert repeating elements into reusable components:
- Navigation bars
- Tab bars
- Cards
- Lists
- Form inputs
- Buttons
- Modals
- Notification elements

4. Detect Layout Logic
Identify:
- Page hierarchy
- Container structure
- Responsive behavior
- Component nesting
- Grid or flex layouts

5. Reconstruct Application Pages
For every page/frame describe:
- Page name
- Purpose of the page
- UI blocks used
- Components on the page
- User actions available

6. Define User Interactions
Identify interactions:
- Button clicks
- Navigation transitions
- Modal openings
- Dropdown selections
- Form submissions
- Hover and focus states

7. Build Navigation Map
Create a clear navigation flow:
- Entry point
- Main navigation
- Page transitions
- Modal flows

8. Prepare Development Structure
Describe the project as a production-ready application:

Frontend structure:
- pages
- components
- layouts
- UI elements

9. Output Format
Provide the result in structured sections:

1. Application Overview
2. Page Structure
3. Navigation Flow
4. Component Library
5. Design System
6. Interaction Map
7. Frontend Architecture

Preserve the exact layout logic and UI structure from the Figma design.
