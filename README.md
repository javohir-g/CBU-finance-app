## 🌟 Визия
**CBU Finance** — это финтех-решение нового поколения для рынка Узбекистана. Мы объединяем удобство Telegram Mini Apps с мощным функционалом банковского приложения, делая управление финансами легким, социальным и прозрачным.

---

## 🔥 Ключевые Фичи (MVP)

### 📈 Интеграция с ЦБ Узбекистана
Забудьте о ручном поиске курсов. Наше приложение напрямую подключено к API Центробанка, предоставляя самые актуальные курсы валют (USD, EUR, RUB и др.) с динамикой изменений в реальном времени.

### 🤝 Социальные накопления: "Копи с друзьями"
Уникальная фича для рынка — создавайте финансовые цели вместе! 
- Планируете поездку с другом?
- Копите на подарок близкому человеку?
- Просто хотите финансовой дисциплины вместе?
**CBU Finance позволяет привязывать партнеров к вашим целям.**

### 📲 Бесшовная авторизация
Никаких паролей, SMS-кодов и сложных регистраций. Используем нативную безопасность Telegram — просто запустите бот и вы уже в системе.

### 💳 Полный контроль карт и трат
- Управление картами **Uzcard** и **Humo**.
- Детальная аналитика расходов в виде графиков.
- История транзакций с автоматической категоризацией.

### 🎁 Система рефералов
Приглашайте друзей через Telegram и получайте бонусы за развитие финансового сообщества Узбекистана.

## 🎨 Дизайн и UX
Мы создали **Premium Experience** для пользователей:
- **Адаптивные темы:** Автоматическое переключение между тёмной и светлой темами вслед за настройками Telegram.
- **Micro-interactions:** Плавные анимации транзакций и прогресса целей.

## 🛠 Технологический Стек (Hackathon Ready)
- **Frontend:** React 18, Vite, Framer Motion (для плавных мобильных UX).
- **Backend:** FastAPI (Python) — для максимальной скорости обработки запросов.
- **Database:** SQLAlchemy & Real-time API Integrations.
- **Integration:** Telegram Mini App API.
